# 這扇門關了，馬上幫你開另一扇窗！

在Lektor中設定自動導向非常地簡單！一樣從model、template、content依序下手：

# Model

models/redirect.ini

在redirect的model中建立target這個字串欄位，指定路徑進行導向。

[model]
name = Redirect

[fields.target]
label = Redirect Target
type = string
description = Target is of type 'string' to allow relative paths. Converted to url in the template.

# Template

template很簡單，一句話：

templates/redirect.html

<meta http-equiv="refresh" content="0; URL='{{ this.target|url }}'" />

# Contents

content/page-to-redirect/contents.lr

在你要進行重新導向的網頁路徑中，修改contents內容，指定model為redirect，讓lektor知道這個網頁需要重新導向。

_model: redirect
---
target: /new/path
---
_discoverable: no

# 自行測試

照著說明做，測試將網頁重新導向到blog頁面，名稱為backToBlog，target就是/blog。

  1. 新增redirect.ini model

  2. 新增redirect.html template

  3. 新增backToBlog/contents.lr，並設定target: /blog

  4. 將backToBlog連結加入nav bar中。

    <nav>
    <ul class="nav navbar-nav">
        <li{% if this._path == '/' %} class="active"{% endif
              %}><a href="{{ '/'|url }}">Welcome</a></li>
        {% for href, title in [
        ['/blog', 'Blog'],
        ['/projects', 'Projects'],
        ['/backToBlog', 'backToBlog'],
        ['/about', 'About']
        ] %}
        <li{% if this.is_child_of(href) %} class="active"{% endif
              %}><a href="{{ href|url }}">{{ title }}</a></li>
        {% endfor %}
    </ul>
</nav>
