# Quick Start 初窺Lektor功能

為了讓第一次接觸的人能快速了解Lektor的功能，Lektor提供了範本，輸入以下命令即可建立範本程式檔。
`$ lektor quickstart`
此時終端機中會引導你輸入相關資訊，建立專案。
